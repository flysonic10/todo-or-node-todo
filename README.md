# Todo or Node Todo

A simple todo application in Express and Angular.

### Instructions
1. Clone the repo  
`git clone git@github.com:flysonic10/todo-or-node-todo.git`

1. Setup environment  
`cp .env.example .env`

1. Install modules  
`npm install`

1. Start app  
`npm start`

1. Do todo as you do  
visit http://localhost:3001/
